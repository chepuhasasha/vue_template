# Локализация

[← Назад к корневому README](../../README.md)

Папка `src/locales/` содержит словари переводов для приложения в формате JSON.

## Структура

- `ru.json` / `en.json` — словари переводов.
- `index.ts` — список локалей, базовая локаль и экспорт словарей.

## Правила

- Все ключи должны присутствовать в каждой локали.
- Ключи хранятся в плоском формате `section.key`.
- Тексты отображаются через `useI18n`.
