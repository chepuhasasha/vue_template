# Страницы

[← Назад к корневому README](../../README.md)

Папка `src/views/` содержит **страницы** (page-level компоненты) приложения.

## Правила структуры

- Каждая страница — отдельная папка в `src/views/`.
- Входной компонент страницы всегда называется `View.vue`.
- Все, что относится только к этой странице (локальные компоненты, композиции, типы, утилиты, стили, данные), хранится внутри папки страницы.
- То, что используется в разных страницах, выносится в общие разделы (`src/components`, `src/composables`, `src/utils`, и т.д.).

Пример структуры страницы:

```
src/views/Profile/
  View.vue
  components/
  composables/
  types/
  assets/
```

## Как добавить новую страницу

1. Создайте папку в `src/views/` (например, `src/views/Profile/`).
2. Добавьте файл `View.vue` внутри папки.
3. Сложите локальные компоненты и композиции в подпапки этой страницы.
4. Подключите страницу в роутере (`src/router.ts`).
